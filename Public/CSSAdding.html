The Cascade in CSS

The Way you've Added CSS Example

The Way you've Added CSS

Inline > Internal > External ! 

      
        

            // External Style.css file
            h1 {
                color: yellow;
            }

External CSS will be applied if only we have it!

The Way you've Added CSS

Inline > Internal > External ! 

      
        

            // External Style.css file
            h1 {
                color: yellow;
            }
        

        // Internal CSS
            <style>
                h1 {
                    color: green;
                }
            </style>

Internal CSS will be applied if we have any conflict property with external Style Sheet!

The Way you've Added CSS

Inline > Internal > External ! 

      
        

            // External Style.css file
            h1 {
                color: yellow;
            }
        

            // Internal CSS
            <style>
                h1 {
                    color: green;
                }
            </style>
        
            // HTML Code
            <h1 style="color:black">Content</h1>

Inline CSS will be applied if we have any conflict property with external or Internal Style Sheets!
