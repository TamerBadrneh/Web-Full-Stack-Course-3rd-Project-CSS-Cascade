Important Example

Importance of that style !

!important CSS notation will applied forever, whatever the environment of conflict was. 

      
      // Inline CSS was the most powerful way to override 
      // a CSS Conflict
      <h1 style="color:black">Content</h1>
      // What about now !
      <style>
        h1 {
            color: green; !important
        }
      </style>

As we mentioned before, whatever the case, Important to CSS means nothing can override it!
